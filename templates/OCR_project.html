Deep OCR with YOLO 3

A deep Learning based OCR solution created using State-of-Art deep learning models YOLO and RCNN.

  • A RCNN is trained to detect ID cards/Documents in complex scenes.

  • Detected document is cropped and adjusted in correct orientation using computer vison techniques.

  • A document classifer was trained with YOLO that classifies documents as Pan, Aadhaar or others. It also classifies different formats of each documents.

  • Different YOLO models are trained for PAN and aadhaar card to detect and classify various entites in the document.

  • Each entity is cropped and passes through tessaract OCR engine and text preprocessing to get clean text output.

  • The model has demonstrated good results even in challanging images. It can handle ID cards in complex backgrounds, wrong orientation and even works on photocopies.

Document Detection

A Tensorflow RCNN model is trained to detect ID cards/documents even in complex and challenging backgrounds.

Document Classification and Adjustments

A YOLO object detection model is trained to detect document specific landmarks to classify documents as PAN, AADHAR, Driving License, Bank Statement etc. Document orientation is also adjusted with Computer Visiontechniques.

Entity Detection with YOLO

Seperate YOLO models are trained for each type of document to detect and classify various entites in the document like Name, ID number, DOB etc. These entites are cropped, prerpocessed and passed to Tessaract OCR engine to get text data.
